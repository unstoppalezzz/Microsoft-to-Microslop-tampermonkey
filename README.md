# Microsoft → Microslop (Tampermonkey Script)

Replaces every visible instance of **Microsoft** with **Microslop** on any website.

## Installation

1. Install the **Tampermonkey** browser extension.
2. Open the **Tampermonkey Dashboard**.
3. Go to the **Utilities** tab.
4. Under **Import from URL**, paste the link below:

https://github.com/unstoppalezzz/Microsoft-to-Microslop-tampermonkey/raw/refs/heads/main/Microsoft%20%E2%86%92%20Microslop.user.js


5. Click **Import**.
6. Tampermonkey will open the script — click **Install**.

# What it Does

- Replaces all visible text **Microsoft** → **Microslop**
- Works on most websites
- Runs automatically after installation
